# idx

## Prerequisites

- [Go >= 1.19](https://golang.org/doc/install)

## Installation

```
go install github.com/choraio/server/cmd/idx@latest
```

## Usage

```
idx help
```

## Example

```
idx postgres://postgres:password@localhost:5432/postgres?sslmode=disable localhost:9090 chora-local 1
```
