chora api

  • post auth login: https://api.chora.io/auth/login

  • get idx group vote: https://api.chora.io/idx/{chain-id}/group-vote/{proposal_id}/{voter}
  • get idx group votes: https://api.chora.io/idx/{chain-id}/group-votes/{proposal_id}
  • get ipfs: https://api.chora.io/ipfs/{cid}
  • post ipfs: https://api.chora.io/ipfs
